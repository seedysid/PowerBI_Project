Data Jobs Market Analysis Dashboard (Power BI)
Project Overview

This project analyzes the 2024 data jobs market using Power BI. The dashboard combines salary, hiring, and geographic information from a real-world dataset of data-related job postings to provide insights into compensation trends, hiring demand, and job distribution.

The objective was to build an interactive dashboard that enables users to explore the job market through dynamic filtering and detailed drill-down analysis.

Dashboard File

Power BI Dashboard: Data_Jobs_Dashboard.pbix

Skills Demonstrated
Data cleaning and transformation using Power Query
Data modeling and calculated measures in Power BI
KPI creation for salary and job market analysis
Interactive dashboard design using slicers, bookmarks, and drill-through
Geographic visualization using map charts
Business storytelling through dashboards
Dashboard Features
Executive Overview

This page provides a summary of the overall data job market, including:

Total job postings
Median yearly salary
Median hourly salary
Top hiring job titles
Global distribution of job postings
Hiring trends over time
Job-Level Analysis

The second page enables detailed exploration of individual job roles by displaying:

Salary distribution
Work-from-home availability
Degree requirements
Hiring platforms
Geographic job locations

Users can navigate directly from the overview page using Power BI drill-through functionality.

Outcome

This project demonstrates how Power BI can transform raw recruitment data into an interactive business intelligence solution. The dashboard helps users identify hiring trends, compare salaries across job roles, and explore the global data job market through intuitive visualizations.
